🚀 My Dev Blog (CRUD Project)
A professional web application for managing blog posts, built with a focus on clean code and great design.

✨ Key Features
Full CRUD Support: Create, read, update, and delete posts easily.

Local Storage: Your posts are saved automatically in the browser, so they don't disappear on refresh.

Live Search: A fast search bar to filter posts by their titles.

Responsive Design: Optimized for all screens (Mobile, Tablet, and Desktop).

Dark Theme: A sleek, modern UI with smooth CSS transitions and hover effects.

🛠️ Built With
HTML5 for structure.

CSS3 for custom styling and responsive layouts.

JavaScript (ES6+) for all logic and data handling.

📖 How to use
Add a title and description for your post.

Click Post to save it.

Use the Search bar to find specific topics.

Click Edit to modify a post or Delete to remove it.
